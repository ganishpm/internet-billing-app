# Internet Billing System

Aplikasi billing internet full-stack yang dibuat dengan Node.js dan MongoDB.

## Fitur

- Manajemen Pelanggan
- Manajemen Paket Internet
- Pembuatan Tagihan Otomatis
- Tracking Pembayaran
- Monitoring PPOE
- WABLAS

## Kebutuhan
- nodejs (version 20) (tested)
- mongodb version 4.4 (tested)
